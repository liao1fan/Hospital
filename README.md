# Hospital

使用教程：
本系统使用C3P0数据库连接池连接Java后端和Mysql数据库，在c3p0-config.xml文件中将密码改为自己数据库的密码即可使用数据库。
在webapp/sql文件夹中找到.sql文件，运行后数据库中出现名为hospital的schemas。
将Java后端在IDEA中配置完成后，启动程序，在网页中访问 "localhost:8080/login1.jsp"进入登录页面。
